# CosPlan 云端同步 — 部署步骤

## 前置条件
- 注册 [Cloudflare](https://dash.cloudflare.com) 账号（免费）
- 安装 [Node.js](https://nodejs.org)（LTS 版本）

## 1. 安装 Wrangler CLI
```bash
npm install -g wrangler
wrangler login
```

## 2. 创建 D1 数据库
```bash
cd worker
wrangler d1 create cosplan-db
```
执行后会输出 database_id，将其填入 `wrangler.toml` 的 `database_id` 字段。

## 3. 初始化数据库表
```bash
wrangler d1 execute cosplan-db --file=schema.sql
```

## 4. 部署 Worker
```bash
wrangler deploy
```
部署成功后会显示 Worker URL，类似：`https://cosplan-api.xxx.workers.dev`

## 5. 配置前端
打开 `cosplan-v2.html`，将顶部的：
```javascript
var API_URL = 'https://cosplan-api.你的域名.workers.dev';
```
改为你实际的 Worker URL。

## 6. 重新部署前端
将修改后的 `cosplan-v2.html` 推送到 GitHub Pages 即可。

## 完成
- 所有设备访问同一个链接，登录同一账号即可同步数据
- 离线时自动降级为本地存储，恢复网络后自动同步
- 免费额度：D1 5GB 存储 + 500万次读取/天，个人使用绰绰有余
