# CosPlan V2.0 - UI 设计规范

> 版本：V2.0 | 日期：2026-05-12 | 主题：明亮简洁

---

## 与 V1.0 的主要差异

| 维度 | V1.0 暗色主题 | V2.0 明亮主题 |
|------|-------------|-------------|
| 背景 | `#1a1a2e` 深紫黑 | `#f5f5f7` 浅灰白 |
| 卡片 | 深色背景区分 | 白色 + 阴影抬升 |
| 文字 | 白色系 | 深色系 |
| 品牌色 | `#9B59B6 / #E91E8C` | `#8B3FAF / #D81B60`（微调对比度） |
| 状态色 | Material Design | iOS 系统色 |
| 层次感 | 亮色卡片浮于暗色背景 | 阴影系统 |
| 视觉风格 | 二次元暗色 | iOS 简洁明亮 |

---

## 1. 设计理念

- **风格**：iOS 风格明亮主题，简洁通透
- **感受**：轻快、干净、现代
- **目标**：减轻视觉疲劳，适合日常高频使用

## 2. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#8B3FAF` | 紫色，品牌主色 |
| 辅色 | `#D81B60` | 粉色，强调/高亮 |
| 渐变 | `#8B3FAF → #D81B60` | 按钮/头部/标签 |
| 背景 | `#f5f5f7` | 浅灰白，iOS 系统背景色 |
| 卡片 | `#ffffff` | 纯白 |
| 卡片悬浮 | `#f0f0f5` | 浅灰 |
| 主文字 | `#1d1d1f` | 近黑色 |
| 次文字 | `#6e6e73` | 中灰 |
| 弱化文字 | `#aeaeb2` | 浅灰 |
| 成功/已完成 | `#34C759` | iOS 绿 |
| 警告/待修 | `#FF9500` | iOS 橙 |
| 原片/待选 | `#FF3B30` | iOS 红 |
| 分割线 | `#e8e8ed` | 浅灰 |

## 3. 阴影系统（V2.0 新增）

| Token | 值 | 用途 |
|-------|-----|------|
| shadow-sm | `0 1px 3px rgba(0,0,0,0.06)` | 卡片默认抬升 |
| shadow-md | `0 4px 12px rgba(0,0,0,0.08)` | 卡片按下态 |
| shadow-lg | `0 8px 24px rgba(0,0,0,0.1)` | 弹窗/底部菜单 |

## 4. 字体

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 22px | 700 | 页面标题 |
| H2 | 18px | 600 | 区块标题 |
| H3 | 16px | 600 | 卡片标题 |
| Body | 14px | 400 | 正文 |
| Small | 12px | 400 | 辅助信息、时间戳 |
| Button | 14px | 600 | 按钮文字 |

## 5. 间距系统

基准：**12px**，所有间距为 12 的倍数。

| Token | 值 | 用途 |
|-------|-----|------|
| xs | 6px | 图标与文字间距 |
| sm | 12px | 卡片内边距、元素间距 |
| md | 18px | 区块间距 |
| lg | 24px | 页面边距、大区块间距 |
| xl | 36px | 页面顶部/底部留白 |

## 6. 圆角

| 元素 | 圆角 | 说明 |
|------|------|------|
| 卡片 | 16px | 项目卡片、清单卡片 |
| 按钮 | 24px | 操作按钮（胶囊形） |
| 输入框 | 12px | 表单输入 |
| 弹窗 | 16px | 模态弹窗 |
| 小标签 | 8px | 状态标签、角标 |
| 图标容器 | 12px | 头像、缩略图 |

## 7. 组件规范

### 7.1 底部导航栏
- 高度：60px + safe-area-inset-bottom
- 均分 5 等份
- 选中态：图标+文字变紫粉渐变，上方有 3px 指示条
- 未选中态：灰色 `#aeaeb2`
- 背景：白色 + 顶部分割线 `#e8e8ed`

### 7.2 统计卡片
- 2×2 网格
- 白色背景 + 阴影 `shadow-sm`
- 数字 22px 粗体，标签 12px 次文字色
- 点击有缩放反馈（scale 0.95）

### 7.3 项目卡片
- 横向布局：左侧头像(48×48) + 右侧信息
- 白色背景 + 阴影 `shadow-sm`
- 进度条：高 4px，圆角 2px，渐变填充
- 状态标签：小圆角标签，颜色对应状态

### 7.4 按钮
- 主按钮：紫粉渐变背景，白字，圆角 24px，高 44px
- 次按钮：透明背景 + 紫边框，紫字
- 危险按钮：红色背景，白字
- 点击态：opacity 0.8 + scale 0.97
- 最小触控区域：44×44px

### 7.5 输入框
- 浅灰背景 `#f5f5f7`，底部 1px 紫边框
- 聚焦：底部边框变粉，加发光阴影
- Placeholder 颜色：`#aeaeb2`
- 高度：44px，圆角 12px

### 7.6 模态弹窗
- 居中显示，背景半透明黑 `rgba(0,0,0,0.55)`
- 内容区：白色，圆角 16px，内边距 24px，阴影 `shadow-lg`
- 关闭按钮：右上角 ×
- 动画：fadeIn + slideUp 0.3s

### 7.7 照片网格
- 3 列，间距 6px
- 每格 1:1 正方形，缩略图 cover 填充
- 状态角标：右下角小圆点(8px)，颜色对应状态

## 8. 动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 页面切换 | fadeIn + slideUp | 0.25s |
| 弹窗打开 | fadeIn + scale(0.9→1) | 0.3s |
| 卡片点击 | scale(1 → 0.96) | 0.15s |
| 按钮点击 | opacity(1 → 0.8) | 0.1s |
| 列表项删除 | slideOut + fadeOut | 0.3s |
| Toast 提示 | slideUp + fadeOut(2s后) | 0.3s |

## 10. 空状态

- 居中显示
- 大号 Emoji (48px) + 提示文字 + 操作按钮
- 文字颜色使用 `--text-muted`

## 11. Toast 提示

- 底部弹出，距底 80px
- 紫粉渐变背景，白字，圆角 24px
- 2.5s 自动消失
- 文字不超过 20 字
