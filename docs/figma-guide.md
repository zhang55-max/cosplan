# Figma 零基础快速入门 — 为 CosPlan 画原型

> 面向纯新手，从零到能画出面试作品，按步骤来

---

## 1. 注册 & 新建

1. 打开 https://www.figma.com
2. 点「Sign up」→ 用 Google 账号或邮箱注册（免费）
3. 进入后点右上角 **「+ New」** → **「Design file」**
4. 白板就绪

---

## 2. 首先理解这三个概念

| 概念 | 是什么 | 干嘛用 |
|------|--------|--------|
| **Frame（画板）** | 手机屏幕大小的白框 | 每页界面放在一个 Frame 里 |
| **Shape（形状）** | 矩形/圆形/线条 | 按钮、卡片、输入框等一切元素 |
| **Text（文字）** | 文字文本 | 标题、正文、按钮文字 |

---

## 3. 第一步：创建手机 Frame

这是最重要的一步，所有内容都画在这个 Frame 里。

**操作：**
- 按键盘 **F** 键
- 右侧面板会出现手机尺寸选项
- 选 **iPhone 15 Pro (393 × 852)**

现在你有一块手机屏幕大小的画布了。

---

## 4. 基础操作（30 分钟练手）

### 4.1 创建矩形（卡片）

| 操作 | 怎么做 |
|------|--------|
| 画矩形 | 按 **R** 键，在画布上拖出一个矩形 |
| 改颜色 | 选中矩形 → 右侧 **Fill** → 点色块 → 选颜色或输入色号 |
| 加圆角 | 选中矩形 → 右侧 **Corner radius** → 输入 `16` |
| 加阴影 | 选中矩形 → 右侧 **Effects** → 点 `+` → 选 Drop shadow |

### 4.2 创建文字

| 操作 | 怎么做 |
|------|--------|
| 添加文字 | 按 **T** 键，在画布点击 → 输入文字 |
| 改字号 | 选中文字 → 右侧 **Text** → 调整数字 |
| 改颜色 | 选中文字 → 右侧 **Fill** → 点色块 |
| 加粗 | 选中文字 → 右侧 **Text** → 点 **Bold** |

### 4.3 创建圆形（头像）

- 按 **O** 键（椭圆工具）
- 按住 **Shift** 拖出一个正圆

### 4.4 创建按钮

- 按 **R** 键画矩形
- 圆角拉到 `24`（胶囊形）
- 添加文字 "登 录"
- 把文字和矩形居中对齐：选中两者 → 上方工具栏点「对齐水平居中」+「对齐垂直居中」

---

## 5. CosPlan 配色表（直接复制用）

在 Figma 右侧 Fill 里输入这些色号：

| 用途 | 色号 | 
|------|------|
| 页面背景 | `F5F5F7` |
| 白色卡片 | `FFFFFF` |
| 标题文字 | `1D1D1F` |
| 次要文字 | `6E6E73` |
| 主色（紫） | `8B3FAF` |
| 辅色（粉） | `D81B60` |
| 边框 | `E8E8ED` |
| 绿色（完成） | `34C759` |
| 红色（危险） | `FF3B30` |

## 6. 画 6 个核心页面（按顺序）

### 页面 1：登录页

按 **F** → 创 iPhone Frame，在里面画：

1. 白色卡片（矩形，宽 320，圆角 16，白色 + 阴影）
2. 标题文字 "CosPlan"，字号 22，加粗
3. 下拉选择器（矩形，圆角 12，边框灰色）
4. 密码输入框（矩形，圆角 12，灰色边框）
5. 按钮（渐变紫色矩形，圆角 24，白字 "登 录"）
6. 小字 "如为新用户请注册"，字号 12，紫色

### 页面 2：首页仪表盘

新建 Frame（按 F 再创一个），画：

1. 顶部标题 "CosPlan ✨"
2. 4 格统计卡片（2×2 网格，白色卡片 + emoji + 数字 + 标签文字）
3. 灵感输入框（圆角矩形 + placeholder 文字）
4. 项目列表（角色头像圆形 + 名称 + 进度条）

### 页面 3：角色详情

1. 大号头像（圆形，80×80）
2. 角色名（字号 22，加粗）
3. 作品来源标签（紫色背景标签）
4. 筹备清单区（带勾选框的列表项）
5. 照片缩略图网格
6. 底部按钮组

### 页面 4：筹备清单

1. 角色选择器（下拉框）
2. 手风琴面板：👗 服装 / 🔧 道具 / 📍 场地 / 🎬 分镜 / 📜 脚本
3. 每项有 checkbox + 标题 + 缩略图
4. 折叠展开箭头

### 页面 5：照片素材

1. 状态筛选标签栏（全部/原片/待修/已修）
2. 3 列照片网格（正方形缩略图）
3. 右下角状态圆点（红/橙/绿）
4. 统计栏

### 页面 6：个人主页

1. 头像 + 昵称 + 编辑图标
2. 3 格统计（角色数/进行中/已完成）
3. 时间线列表（竖线 + 圆点 + 卡片）
4. 底部操作按钮

---

## 7. 加分项：做交互原型

画完页面后，可以做点击跳转：

1. 选中登录页的「登录」按钮
2. 右侧面板点 **Prototype** 标签
3. **Interaction** 点 `+` → 选择 **On click**
4. **Action** 选 **Navigate to** → 选首页 Frame
5. 播放预览：点右上角 ▶️ 按钮

---

## 8. 三个小技巧

| 技巧 | 怎么做 |
|------|--------|
| 快速复制 | 按住 **Alt** 拖动元素 |
| 等距排列 | 选中多个元素 → 右上角点 **Tidy up** |
| 图层管理 | 左侧面板可以拖拽图层上下顺序，给图层改中文名 |
| 导出图片 | 选中 Frame → 右侧 **Export** → 选 PNG 2x |

---

## 9. 半天能做完的程度

时间有限的话，重点做好这 3 个页面就够了面试用：

1. **登录页**（1 个 Frame）
2. **角色详情页**（最能体现产品逻辑的页面，1 个 Frame）
3. **个人主页/时间线**（体现数据思维，1 个 Frame）

做 3 个页面 + 加上点击跳转交互 ≈ 3-4 小时。

---

## 10. 做完后怎么放进作品集

1. Figma 中选中 Frame → 右侧 Export → PNG 2x → 导出
2. 或者直接分享 Figma 链接（右上角 Share → Copy link → 设置 Anyone with the link can view）
3. 把图片/链接放进 `portfolio-brief.md` 或飞书文档

面试时直接打开 Figma 原型链接演示交互，比 PPT 强 10 倍。
