# CosPlan - UI 设计规范

> 版本：V1.0 | 日期：2026-05-12

---

## 1. 设计理念

- **风格**：二次元暗色主题，紫粉渐变，圆润温暖
- **感受**：专业但不冰冷，可爱但不幼稚
- **目标**：COSER 用户一眼就喜欢，愿意日常使用

## 2. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#9B59B6` | 紫色，品牌主色 |
| 辅色 | `#E91E8C` | 粉色，强调/高亮 |
| 渐变 | `#9B59B6 → #E91E8C` | 按钮/头部/标签 |
| 背景 | `#1a1a2e` | 深紫黑，护眼 |
| 卡片 | `#252547` | 半透明深紫 |
| 卡片悬浮 | `#2f2f5a` | 略亮 |
| 主文字 | `#ffffff` | 纯白 |
| 次文字 | `#b0b0cc` | 灰紫 |
| 成功/已完成 | `#4CAF50` | 绿色 |
| 警告/待修 | `#FF9800` | 橙色 |
| 原片/待选 | `#F44336` | 红色 |
| 分割线 | `#333355` | 暗紫 |

## 3. 字体

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 22px | 700 | 页面标题 |
| H2 | 18px | 600 | 区块标题 |
| H3 | 16px | 600 | 卡片标题 |
| Body | 14px | 400 | 正文 |
| Small | 12px | 400 | 辅助信息、时间戳 |
| Button | 14px | 600 | 按钮文字 |

## 4. 间距系统

基准：**12px**，所有间距为 12 的倍数。

| Token | 值 | 用途 |
|-------|-----|------|
| xs | 6px | 图标与文字间距 |
| sm | 12px | 卡片内边距、元素间距 |
| md | 18px | 区块间距 |
| lg | 24px | 页面边距、大区块间距 |
| xl | 36px | 页面顶部/底部留白 |

## 5. 圆角

| 元素 | 圆角 | 说明 |
|------|------|------|
| 卡片 | 16px | 项目卡片、清单卡片 |
| 按钮 | 24px | 操作按钮（胶囊形） |
| 输入框 | 12px | 表单输入 |
| 弹窗 | 16px | 模态弹窗 |
| 小标签 | 8px | 状态标签、角标 |
| 图标容器 | 12px | 头像、缩略图 |

## 6. 组件规范

### 6.1 底部导航栏
- 高度：60px + safe-area-inset-bottom
- 均分 5 等份
- 选中态：图标+文字变紫粉渐变，上方有 3px 指示条
- 未选中态：灰色 `#6a6a8a`

### 6.2 统计卡片
- 2×2 网格
- 深色背景 `#252547`，圆角 16px
- 数字 22px 粗体，标签 12px 次文字色
- 点击有缩放反馈（scale 0.95）

### 6.3 项目卡片
- 横向布局：左侧头像(48×48) + 右侧信息
- 进度条：高 4px，圆角 2px，渐变填充
- 状态标签：小圆角标签，颜色对应状态

### 6.4 按钮
- 主按钮：紫粉渐变背景，白字，圆角 24px，高 44px
- 次按钮：透明背景 + 紫边框，紫字
- 危险按钮：红色背景
- 点击态：opacity 0.8 + scale 0.97
- 最小触控区域：44×44px

### 6.5 输入框
- 深色背景 `#1a1a2e`，底部 1px 紫边框
- 聚焦：底部边框变粉，加发光阴影
- Placeholder 颜色：`#6a6a8a`
- 高度：44px，圆角 12px

### 6.6 模态弹窗
- 居中显示，背景半透明黑 `rgba(0,0,0,0.7)`
- 内容区：`#252547`，圆角 16px，内边距 24px
- 关闭按钮：右上角 ×
- 动画：fadeIn + slideUp 0.3s

### 6.7 照片网格
- 3 列，间距 6px
- 每格 1:1 正方形，缩略图 cover 填充
- 状态角标：右下角小圆点(8px)，颜色对应状态

## 7. 动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 页面切换 | fadeIn + slideUp | 0.25s |
| 弹窗打开 | fadeIn + scale(0.9→1) | 0.3s |
| 卡片点击 | scale(1 → 0.96) | 0.15s |
| 按钮点击 | opacity(1 → 0.8) | 0.1s |
| 列表项删除 | slideOut + fadeOut | 0.3s |
| Toast 提示 | slideUp + fadeOut(2s后) | 0.3s |

## 9. 空状态

- 居中显示
- 大号 Emoji (48px) + 提示文字 + 操作按钮
- 例：「📁 还没有角色项目，点击下方按钮创建第一个吧！」

## 10. Toast 提示

- 底部弹出，距底 80px
- 紫粉渐变背景，白字，圆角 24px
- 2.5s 自动消失
- 文字不超过 20 字
