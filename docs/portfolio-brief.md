# CosPlan — 作品集展示页

> COSER 创作助手 | 2026.05 | 独立产品负责人

---

## 一句话介绍

为 COSER 打造的一站式角色项目管理工具，覆盖筹备→拍摄→选片→修图→发布全流程。

## 在线体验

🔗 https://zhang55-max.github.io/cosplan/

## 解决什么问题

COSER 筹备一个角色需要：买服装道具、定场地、画分镜、写脚本，信息散落在微信/淘宝/备忘录/相册中。拍摄后几百张原片需要筛选标记，和后期沟通修图需求全靠微信刷屏。

CosPlan 把这些全部整合在一个工具里。

## 核心功能

| 模块 | 功能 |
|------|------|
| 📁 角色管理 | 创建角色项目，上传头像/参考图，5 阶段状态流转 |
| 📋 筹备清单 | 服装/道具/场地/分镜/脚本 五分类，支持多图、链接、备注 |
| 🖼️ 照片素材 | 上传、压缩、三状态标记（原片/待修/已修），下载原图到相册 |
| ✍️ 文案生成 | 台词库 + 标签库（多平台），一键生成发布文案 |
| 💡 灵感速记 | 快速记录，编辑历史保留，每条有独立时间戳 |
| 📅 时间线 | 角色项目从创建到完成的完整时间记录，可作为原创证明 |
| 👤 个人主页 | 头像/昵称/密码/账号管理/数据统计 |

## 产品亮点

- **零学习成本**：按 COSER 自然工作流设计，无需教程
- **离线可用**：纯本地运行，拍摄现场没网络也能用
- **多用户**：支持多人共用一台设备，数据互不干扰
- **原创证明**：自动时间线记录每个操作节点，不可篡改
