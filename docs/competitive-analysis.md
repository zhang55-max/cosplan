# CosPlan — 竞品分析

> 日期：2026-05 | 版本：V1.0

---

## 1. 市场现状

COSER 群体在国内约 200-300 万人（含泛二次元），核心活跃 COSER 约 50-80 万。目前**没有一款产品专门服务 COSER 的「项目制」工作流**，大多数人用通用工具拼凑。

## 3. 差异化分析

### CosPlan 的独特价值

| 维度 | 现有方案 | CosPlan |
|------|---------|---------|
| **角色项目制** | ❌ 无此概念，信息散落各处 | ✅ 以角色为中心，串联全流程 |
| **一站式** | ❌ 需要 4-5 个工具切换 | ✅ 一个 App 覆盖筹备→发布 |
| **时间线原创证明** | ❌ 无此功能 | ✅ 自动记录每步操作 + 时间戳 |
| **离线可用** | ❌ 多数需要网络 | ✅ 纯本地，无网络依赖 |
| **多用户** | ❌ 不支持 | ✅ 多人共用一台设备，数据隔离 |
| **移动端优先** | ⚠️ 部分支持 | ✅ 触控 44px+，手机原生体验 |
| **COSER 专属文案** | ❌ 通用模板 | ✅ 台词库+标签库，一键拼接 |
| **无学习成本** | ⚠️ 需学工具用法 | ✅ 照 COSER 自然工作流设计 |

## 4. 市场定位

```
               高结构化
                  │
       Notion      │    CosPlan ★
       Trello      │
       Teambition  │
                  │
  ────────────────┼────────────────
                  │
   微信收藏       │    手机相册
   备忘录         │    手机备忘录
   淘宝购买记录   │
                  │
               低结构化
                  
      通用工具 ←────────→ COSER 垂直
```

CosPlan 定位：**COSER 垂直领域的轻量级项目管家**，填补通用工具（太乱）和重型项目管理（太重）之间的空白。

## 5. SWOT 分析

### 优势（Strengths）
- 精准解决 COSER 真实痛点
- 零依赖，离线可用，隐私安全
- 时间线原创证明是差异化杀手锏
- 移动端优先，上手零门槛

### 劣势（Weaknesses）
- 纯前端，无法跨设备自动同步
- 单人开发，资源有限
- 依赖浏览器，非原生 App

### 机会（Opportunities）
- COSER 群体增长中（二次元文化主流化）
- 无直接竞品，蓝海市场
- 可扩展：摄影师版、后期版、社团版
- 微信小程序化可触达更多用户

### 威胁（Threats）
- 大厂可能切入（小红书已覆盖发布环节）
- 微信小程序生态内可能出现竞品
- 用户习惯难以改变（从微信迁移有成本）

## 6. 核心指标（预期）

| 指标 | 目标 | 衡量方式 |
|------|------|---------|
| 筹备完成率提升 | +30% | 筹备项完成数/总数 |
| 选片效率 | 减少 50% 时间 | 三状态标记替代手动分类 |
| 文案输出速度 | 3 秒生成 | 一键拼接台词+标签 |
| 用户留存 | 周留存 60%+ | 角色项目制天然促留存 |
